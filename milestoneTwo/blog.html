Project 2 - Milestone 1

Description

Our single page application (SPA) is to be a fun trivia game that gives the player different trivia questions within different categories. The inspiration for this SPA is the game Trivia Crack that became extremely popular around 2014. In Trivia Crack players would spin the wheel to get the question category, if they got the question correct they would get to spin the wheel for a new question. The goal was to get a character, which was a symbol of category mastery, before the other person. Our website will draw similar features from this game. We intend to have different question categories and present the user with different types of trivia questions. Our website will be different in key ways as ours is going to be single player and track overall success/progress instead of collecting icons. Our audience will most likely be ages 16-35 as we intend to find an API that has cultural trivia meant for young adults and teens.

Sources

For this project we are going to use a trivia API to generate all of the questions. The API that we chose is Open Trivia Database. This API is free and does not require an API key or account. This API also allows us to pull more than just the question and answer, it has other false answer options, a difficulty level and a category. This will allow us to potentially track the “expert” level of the player as well.

Stretch Goals

  • Tracking progress by question category.
